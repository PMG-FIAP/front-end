Perguntas Frequentes

Como posso reportar um alagamento?

Para reportar um alagamento, acesse a página do mapa, navegue até a localização desejada e clique no ponto onde está ocorrendo o alagamento. O sistema automaticamente registrará a localização e o horário do reporte.

O que significam as cores no mapa?

As áreas em amarelo indicam regiões com até 3 reportes de alagamento. Áreas em vermelho indicam regiões com mais de 3 reportes, representando pontos mais críticos que requerem maior atenção.

Os dados são atualizados em tempo real?

Sim, os dados são atualizados em tempo real. Cada vez que um novo alagamento é reportado, o mapa é atualizado automaticamente, recalculando as áreas de risco.

Posso editar ou remover um reporte?

Atualmente, o sistema não permite a edição ou remoção de reportes. Isso é uma medida de segurança para manter a integridade dos dados. Se você notar alguma informação incorreta, entre em contato conosco.

Como o sistema calcula as áreas de risco?

O sistema agrupa os reportes que estão em um raio de 300 metros. Se houver mais de 3 reportes em uma mesma área, ela é marcada como área de alto risco (vermelho). Caso contrário, é marcada como área de atenção (amarelo).

Os dados são compartilhados com órgãos públicos?

Atualmente, os dados são armazenados localmente no seu navegador. Não há compartilhamento automático com órgãos públicos. Esta é uma versão inicial do sistema, focada em demonstrar o conceito.
